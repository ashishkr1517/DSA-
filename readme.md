## 🛠️ Getting Started

To compile and run a C++ file:

```bash
# Compile
g++ filename.cpp -o output

# Run
./output
```

Make sure you have `g++` installed. You can install it using:

```bash
# For Debian/Ubuntu
sudo apt install g++

# For Windows (via MinGW)
https://www.mingw-w64.org/
````
